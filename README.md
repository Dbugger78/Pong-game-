# Pong-game

Hello welcome to my pong game in python using turtle import

To run this I highly advise you to use an IDE or modern text editor like VS code or Jet brains however a simple terminal or the basic python shell works perfectly fine.
Python comes mnanually with python so there is no externally importing 3rd party librarys.To run this you should use python verison 3 and onwards however version 2 may work.

### Installation

To install python you can either visit https://www.python.org/downloads/

or type one of these comands into your temrinal

## Installation

* **Mac OS:**
```bash
brew update
brew install python@3.14
```

* **Linux:**
```bash
sudo apt update
sudo apt install python3.14
```

* **Windows:**
```powershell
winget install Python.Python.3.14
```
